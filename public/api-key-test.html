Implementation Details

🔑 Key Generation

  • • Format: ak_live_[64-character-hex]
  • • Security: crypto.randomBytes(32)
  • • Auto-generated: On user registration
  • • User-specific: Tied to account ID
  • • Permissions: authorize, confirm, refund
  • • Tracking: Usage count & last used

🔄 Rotation & Management

  • • Instant rotation: Old key revoked immediately
  • • Zero downtime: New key available instantly
  • • Name preservation: New key keeps same name
  • • Manual revocation: Immediate deactivation
  • • Audit trail: Creation & revocation timestamps
  • • Limits: Max 10 keys per user

⚡ Rate Limiting Details

Limit: 1000 requests/hour per key
Response: 429 status when exceeded
Reset: Rolling window, per-key isolation

API Endpoints Available

GET /api/keys

List all API keys for authenticated user

POST /api/keys

Create new API key with name

POST /api/keys/{keyId}/rotate

Rotate API key (revoke old, create new)

DELETE /api/keys/{keyId}

Revoke API key permanently

GET /api/keys/{keyId}/usage

Get usage statistics and rate limit status
