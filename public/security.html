JWT Security Architecture

Token Structure

// Header
{
"alg": "HS256",
"typ": "JWT"
}
// Payload
{
"sub": "user_id",
"sessionId": "uuid",
"permissions": ["authorize"],
"exp": timestamp,
"iat": timestamp
}
// Signature
HMACSHA256(base64(header) + "." + base64(payload), secret)
